## Probe Rules

- Prefer fixture/local helper probes over live external API/GPU calls.
- Expensive or external operations pass if the system prepares correct artifacts and confirmation boundary.
- Generated skill text is evidence because it is the user-facing Codex contract.
- A scenario can pass with a deferred/companion route if the docs and recommendations are honest about that route.

## Scenario Pass Gate

Round 1 must inspect all S01-S15 at least statically. Dynamic probes are required for S01, S02, S04, S09, S10, S13, and S14 unless an equivalent existing test already covers the exact behavior.
