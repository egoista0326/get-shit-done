# 01-02 Plan: Subagent Review Round 1 - Source Coverage And Index Completeness

## Objective

Run the first review round over the initial Phase 01 framework artifacts, focused on whether all required source surfaces were covered and whether source indexes are precise enough for future upgrades.

## Scope

Review only phase-local framework artifacts, subagent reports in the conversation, and targeted source paths needed to validate missing or suspicious citations. This is a review round, not another broad extraction pass.

## Required Outputs

- `.planning/phases/01-source-framework-extraction/01-REVIEW-R1-SOURCE-COVERAGE.md`
- `.planning/phases/01-source-framework-extraction/01-REVIEW-R1-FIXES.md`
- Updated initial framework artifacts if accepted findings require correction

## Reviewer Lanes

- Upstream GSD source coverage reviewer.
- ljx-GSD/history source coverage reviewer.
- Auto/ARIS source coverage reviewer.
- Source-index precision reviewer.
- Prompt-body locator reviewer.
- Context hygiene reviewer.

If only six subagent slots are available, run exactly these six lanes. If fewer are available, batch the remaining review lanes later rather than letting the main agent substitute.

## Execution Steps

1. Spawn review subagents with phase-local artifacts and lane-specific source scopes.
2. Require each reviewer to return findings with severity, evidence, affected artifact, and concrete correction.
3. Main agent confirms each finding against the artifact and source citation.
4. Accept only findings that materially affect coverage, source traceability, or future framework usability.
5. Apply accepted corrections serially to phase-local framework artifacts.
6. Write the round report and fix plan.

## Acceptance Criteria

- Review round 1 produces a source coverage report and fix plan.
- All accepted source/index findings are applied or explicitly deferred with rationale.
- Main-agent confirmation is limited to checking reviewer findings and applying fixes; it does not perform missing lane extraction.
- No implementation edits are made.
