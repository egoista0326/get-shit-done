# 18-03: Implement Accepted Fixes

<objective>
Repair confirmed Round 1 bugs with minimal changes while preserving GSD and Auto/ARIS capability.
</objective>

## Tasks

- For each confirmed bug, identify the root cause before editing.
- Add or update targeted tests where the behavior is testable.
- Patch the smallest implementation surface that owns the bug.
- Regenerate preview output when generator/runtime/source changes require it.
- Avoid unrelated refactors.

## Verification

- Targeted tests pass.
- Full baseline verification is rerun when feasible.
- Generated preview output matches source/manifest/helper behavior.
