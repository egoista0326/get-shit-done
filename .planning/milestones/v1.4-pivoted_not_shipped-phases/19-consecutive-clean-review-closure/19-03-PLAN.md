# 19-03: Repeat Bounded Review Until Two Clean Rounds

<objective>
Continue fresh review rounds after Round 2 until two consecutive clean rounds pass or the 11-round cap is reached.
</objective>

## Tasks

- If Round 2 is clean, run Round 3 immediately as an independent repeat review.
- If a bug is fixed, reset the clean streak to 0 and continue with the next round.
- Keep each round documented as `ROUND-XX-REVIEW.md` plus confirmation notes if candidates appear.
- Stop at Round 11 if two consecutive clean rounds are not achieved.

## Verification

- Review loop state accurately records current round, clean streak, and stop reason.
- Two clean rounds are consecutive, not merely total.
