   - pause/resume friendliness
2. Define the shared command contract for all top-level lifecycle commands:

4. Record how direct research workflows interact with lifecycle status so `next` can keep working after manual expert use.
5. Note any remaining open points that should be handled in Phase 5 or Phase 6 rather than buried in command docs.
